# 番茄小说自动生产 Agent

这是一个可安装的 Codex 插件市场，包含 `fanqie-novel-production-agent`。

## 安装

```bash
codex plugin marketplace add kato1125/fanqie-novel-production-agent
```

重启 ChatGPT/Codex 桌面端，在插件目录中选择“番茄小说 Agent”，安装“番茄小说自动生产 Agent”。

新建任务后调用：

```text
$fanqie-novel-production-agent
```

首次运行会要求创建使用者自己的 `番茄小说Agent配置.json`，包括作者名、输出目录、GPTs 地址、非敏感账号提示和可选 Word 模板。

本仓库不包含任何人的密码、Cookie、Token、浏览器登录状态、番茄草稿或真实个人配置。
